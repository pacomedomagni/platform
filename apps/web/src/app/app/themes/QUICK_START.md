# Quick Start Guide - Theme Management UI

Get up and running with the NoSlag theme management interface in 5 minutes!

## üöÄ Getting Started

### 1. Navigate to Themes

```
/app/themes
```

You'll see the theme gallery with all available themes.

### 2. Create Your First Theme

**Click "Create New Theme" button**

Choose one of three options:

#### Option A: Start from Preset (Recommended)
```
1. Click "Start from Preset"
2. Select a preset (Modern, Minimal, Bold, or Elegant)
3. Enter a name (e.g., "My Store Theme")
4. Click "Create Theme"
5. Opens in customizer automatically
```

#### Option B: Start from Scratch
```
1. Click "Start from Scratch"
2. Enter name and description
3. Click "Create Theme"
4. Opens in customizer with default settings
```

#### Option C: Import Theme
```
1. Click "Import Theme"
2. Paste theme JSON
3. Enter name and description
4. Click "Create Theme"
```

### 3. Customize Your Theme

The customizer opens with two panels:

**Left Panel**: Settings
**Right Panel**: Live Preview

#### Quick Customization Checklist

```
‚úÖ Colors (5 mins)
   - Pick primary color
   - Pick secondary color
   - Pick accent color

‚úÖ Typography (3 mins)
   - Choose body font
   - Choose heading font
   - Adjust font sizes

‚úÖ Layout (2 mins)
   - Set layout style
   - Adjust spacing
   - Set container width

‚úÖ Components (2 mins)
   - Button style
   - Card style
   - Input style

‚úÖ Product Display (1 min)
   - Grid columns
   - Image ratio
   - Toggles
```

### 4. Save and Activate

```
1. Press Cmd+S (or click Save button)
2. Close customizer (Esc or X)
3. Back in gallery, click "Activate" on your theme
4. Done! Your storefront is updated
```

## üéØ Pro Tips

### 1. Use Recent Colors

After picking a color, it's saved to "Recent" swatches. Reuse for consistency!

### 2. Font Preview

Font selector shows real preview. Type to search! Try: "Inter", "Roboto", "Playfair"

### 3. Live Preview

Changes appear instantly in preview. No need to save to see results.

### 4. Unsaved Changes

Yellow dot appears on Save button when you have unsaved changes.

### 5. Reset Colors

Each color picker shows contrast recommendations for accessibility.

### 6. Custom CSS

Advanced users: Add custom CSS in the "Advanced" section for full control.

### 7. Duplicate First

Before making major changes, duplicate the theme as backup.

### 8. Name Convention

Use descriptive names: "Summer Sale 2024" instead of "Theme 1"

## üêõ Troubleshooting

### Theme not saving?

```
- Check internet connection
- Look for error toast notification
- Try again in a few seconds
- Check browser console for errors
```

### Preview not updating?

```
- Click Refresh button in preview toolbar
- Wait for debounce (300ms)
- Check if iframe loaded properly
```

### Colors look wrong?

```
- Check contrast checker recommendations
- Test in different lighting
- View on different devices
- Consider accessibility guidelines
```

### Fonts not loading?

```
- Check internet connection (Google Fonts CDN)
- Try different font
- Clear browser cache
- Refresh page
```
